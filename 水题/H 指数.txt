题目难度：MEDIUM

给定一位研究者论文被引用次数的数组（被引用次数是非负整数）。
编写一个方法，计算出研究者的 h 指数。
h 指数的定义：
h 代表“高引用次数”（high citations），一名科研人员的 h 指数是指他（她）的 （N 篇论文中）总共有 h 篇论文分别被引用了至少 h 次。
且其余的 N - h 篇论文每篇被引用次数 不超过 h 次。
例如：
某人的 h 指数是 20，这表示他已发表的论文中，每篇被引用了至少 20 次的论文总共有 20 篇。
示例：
输入：citations = [3,0,6,1,5]
输出：3 
解释：
给定数组表示研究者总共有 5 篇论文，每篇论文相应的被引用了 3, 0, 6, 1, 5 次。
由于研究者有 3 篇论文每篇 至少 被引用了 3 次，其余两篇论文每篇被引用 不多于 3 次，所以她的 h 指数是 3。

思路：双指针
先对给定数组排序
然后双指针分别从0及数组最后一项对应下标开始分别右移和左移
循环条件为左边指针值要大于数组右边指针对应下标的数组值（保证总共有h篇论文分别被引用了至少h次且其余的N-h篇论文每篇被引用次数不超过h次）
最后返回左边指针的值即可

class Solution {
public:
    int hIndex(vector<int>& citations) {
        sort(citations.begin(), citations.end());
        int res = 0, len = citations.size() - 1;
        while(res < citations.size() && res < citations[len]) {
            res++;
            len--;
        }
        return res;
    }
};
